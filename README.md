# Frontend Blog

## Projeto para praticar React.js usando TypeScript


### Para rodar a aplicação:
 * Use o comando:
    - ```git clone https://github.com/Itor-Carlos/frontend-blog.git```
 * Abra um terminal dentro da pasta que surgiu após o comando anterior
 * Rode o comando:
    - ```yarn```
 * Rode o comando:
    - ```yarn dev```
 * Aproveite :)
